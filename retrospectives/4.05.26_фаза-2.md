- `OPENROUTER_API_KEY` всегда нужен для шагов с benchmark — держать в `.env`, агенту передавать инструкцию загрузки из `.env`
- После каждой фазы: `npm run plans:check:strict` + `npm run build` (если TypeScript затронут)
