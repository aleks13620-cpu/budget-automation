# MVP — что уже собрано и работает

## Статус

MVP реализован и развёрнут в продакшн (`5.42.103.63`). Ниже — то, что работает прямо сейчас.

## Ключевой сценарий (happy path)

1. Создать проект
2. Загрузить Excel-спецификацию → система определяет раздел, заголовок, колонки
3. Загрузить счета поставщиков (PDF или Excel) → система парсит позиции
4. Запустить сопоставление → алгоритм строит пары с confidence score
5. Оператор подтверждает / отклоняет спорные пары
6. Посмотреть итоги по разделам
7. Экспортировать итоговую спецификацию в Excel

## Что ещё не реализовано (дыры)

- Аутентификация пользователей (нет совсем)
- Мультитенантность
- Email-уведомления
- API для внешних интеграций (только внутренний UI)
- Автоматический запуск сопоставления при загрузке (нужен ручной запуск)
