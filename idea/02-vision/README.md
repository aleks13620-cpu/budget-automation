# Видение — что уже реализовано

## Реализованный функционал

### Управление проектами
- Создание, редактирование, удаление проектов
- Каждый проект объединяет спецификации и счета

### Загрузка и парсинг спецификаций
- Загрузка Excel-файлов спецификации (одиночная и массовая)
- Автоматическое определение раздела (8 разделов: электрика, ВК, вентиляция, пожаротушение, КИПиА, слаботочка, АИИС, связь)
- Автоопределение заголовочной строки и колонок по ключевым словам
- Ручная переконфигурация маппинга колонок через UI
- GigaChat-обогащение: дозаполнение пропущенных полей (характеристики, производитель, артикул)
- История версий + откат к предыдущей версии

### Загрузка и парсинг счетов
- PDF (извлечение таблиц + fallback на GigaChat)
- Excel/XLSX (с определением листа, заголовка, колонок)
- Изображения (JPEG, PNG, TIFF — через GigaChat)
- Массовая загрузка (параллельно, 3 потока)
- Ручной ввод позиций (для нечитаемых документов)
- Пропуск нечитаемых документов с пометкой
- История версий + откат

### Конфигурация парсера
- Предпросмотр сырых таблиц из документа
- Визуальный маппинг колонок (какая колонка — название/артикул/цена и т.д.)
- Сохранение конфигурации на уровне поставщика — применяется автоматически при следующей загрузке

### Сопоставление позиций
- Запуск алгоритма в полном или инкрементальном режиме
- 4-уровневый алгоритм совпадений с confidence score
- Tier 5: валидация слабых совпадений через GigaChat
- Подтверждение / отклонение / выбор лучшего совпадения
- Массовые операции (bulk confirm/reject/select)
- Ручное создание совпадения (выбор из поиска)
- Импорт правил из Excel

### Обучение и накопление знаний
- Сохранение подтверждённых совпадений как правил (matching_rules)
- Автоматическое изучение строительных синонимов из подтверждённых пар
- Конфигурации парсера по поставщику
- Кэш результатов GigaChat по хэшу файла

### Ценообразование
- Учёт НДС per-supplier (ставка + включена/не включена)
- Расчёт цены из суммы и количества
- Пользовательская формула расчёта цены
- Применение скидки к счёту
- Раздельный учёт доставки

### Итоги и экспорт
- Раздельные итоги по разделам спецификации
- Три режима экспорта: лучшая цена / оригинальный поставщик / аналоги
- Excel-файл с группировкой по разделам и итоговыми строками

### Обратная связь
- Форма ошибки оператора (к конкретной позиции или счёту)
- Общий список всех ошибок по проектам
- Экспорт ошибок в Excel
- Статус ошибки (новая / решена)
