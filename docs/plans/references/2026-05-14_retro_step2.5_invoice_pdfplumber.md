## Ключевые баги и решения

| Баг | Причина | Решение |
|-----|---------|---------|
| Цены в 100x | `parse_number` regex `[руб₽р.]` удалял десятичную точку | Якорный суффиксный regex `\s*(?:руб\|₽\|р)\.?\s*$` |
| Metadata: bank account вместо invoice# | `\xa0` (non-breaking space) в тексте, regex с literal spaces | Нормализация `\xa0→space` перед regex |
| НЗВЗ: 68/153 позиций | Маленькие таблицы (1-2 строки) отбрасывались как stamp | `is_stamp_table` skip при наличии `prev_mapping` |
| НЗВЗ: 89/153 позиций | Standalone header маппинг (5 cols) vs данные (6 cols) | Data inference приоритетнее header mapping |
| Empty fullText → category C | `categorizeParsingResult` требует текст ≥50 символов | Конструируем fullText из item names |
| File handle leaks | `fitz.open()`/`pdfplumber.open()` без `with` | Context managers |
