# Продукт: обзор

## Название
Система автоматизации расчётов для инженерных сетей

## Суть в одном предложении
Веб-система, которая автоматически сопоставляет позиции из проектной спецификации со счетами поставщиков, проставляет цены и формирует итоговый отчёт по разделам для подготовки бюджета.

## Тип
Веб-приложение (работает в браузере, доступ через интернет)

## Текущий статус разработки
Фазы 3–4 из 9. Есть: загрузка и распознавание, ручной ввод/коррекция, модуль сравнения.

## Стек
- Backend: Node.js + TypeScript
- БД: SQLite
- LLM для парсинга: GigaChat-2 (российский)
- Хостинг: Timeweb
- Репозиторий: GitHub (aleks13620-cpu/budget-automation)
- Разработка: Cursor / VS Code + Claude Code

## Этапы (план)
- **MVP (Этап 1):** расчёт затратной части по материалам — по разделам + общий итог
- **Этап 2:** закупка, логистика, склад, объектный учёт
- **Этап 3+:** интеграция с AMO CRM, 1С, мобильное приложение
