# Каналы привлечения (для строительной компании Артёма)

## Текущие каналы

| Канал | Описание |
|-------|----------|
| Сарафан | Основной исторический канал. 80–90% сделок раньше приходило через Артёма лично. |
| Физический поиск | «Ножками»: увидел кран → зашёл → взял паспорт объекта → добиваться встречи. |
| Тендеры | Мониторинг тендеров. Увидел кто выиграл → пришёл к генподрядчику как субподрядчик. |
| Поставщики | Поставщики сами приносят информацию об объектах. |
| Застройщики | Сайты новостроек, прямые обращения. |
