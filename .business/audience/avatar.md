# Портрет клиента (аватар)

## Основной заказчик: Артём

- **Роль:** Управленец / владелец строительной компании по инженерным сетям
- **Профиль:** Нетехнический. Принимает бизнес-решения, не занимается разработкой.
- **Главная боль:** Не может масштабировать продажи — потолок 4–5 проектов/мес из-за ручного расчёта
- **Что хочет видеть:** Итог по разделам + общий итог. Минимум ручного труда.
- **Язык общения:** Простой, без технического жаргона
- **Критерий успеха:** Берёт цифры → вставляет в бюджет → выпускает КП. Быстро.

## Конечные пользователи системы

| Пользователь | Квалификация | Задача в системе |
|---|---|---|
| Оператор/студент | Низкая | Загружает счета, подтверждает сопоставления |
| Снабжение | Средняя | Работает со спецификацией, контролирует качество |
| Артём | Управленец | Смотрит итоговый отчёт, принимает решения |
| Сметчик | Специалист | Добавляет стоимость работ (будущий этап) |
