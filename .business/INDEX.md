# .business/ - Единый бизнес-контекст клиента

Папка содержит структурированную информацию о бизнесе заказчика системы budget-automation.
Это единственный источник бизнес-контекста в проекте.

## Структура
- company/ - о компании: профиль, команда, юридическая информация, принципы
- products/ - описание продукта/решения, MVP-функции, ценообразование
- audience/ - портрет клиента, сегменты, путь пользователя, возражения
- economics/ - выручка, затраты, unit-экономика, прогноз
- goals/ - KPI и цели по горизонтам
- marketing/ - каналы, воронка, конкуренты, позиционирование
- assets/ - кейсы, отзывы, бренд-материалы

## Правило использования
Любой агент, которому нужен бизнес-контекст, читает этот файл первым, затем профильные файлы разделов.
При появлении новых данных объединять их в структуру `.business` без создания дублирующих папок.
