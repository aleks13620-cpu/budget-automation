# .business/ — Контекст бизнеса клиента

Папка содержит описание бизнеса заказчика системы budget-automation.
Заполняется отдельно — не в рамках этой задачи.

## Структура
- company/   — о компании: название, чем занимаются, команда
- products/  — что продают: инженерные сети, типы объектов
- audience/  — клиенты и объекты (5000м²+)
- economics/ — экономика: потолок проектов, боли, цели автоматизации

## Правило
Любой агент, которому нужен бизнес-контекст, читает этот файл первым.
Не заполнять программно — только человек вручную или отдельный агент с записью разговора.
