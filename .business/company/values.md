# Ценности и принципы

## Из контекста

- **Скорость** — критично сократить время расчёта с 1–1.5 недели до 1–2 дней
- **Масштабируемость** — снять потолок по количеству проектов без роста штата
- **Простота для оператора** — «получил счёт → загрузил → всё»
- **Контроль качества** — окончательное решение всегда за человеком (human-in-the-loop)
- **Реализм** — не обещать 100% автоматизации там, где это невозможно
