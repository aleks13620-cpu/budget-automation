# Команда

## Ключевые роли (по процессу)

| Роль | Функция |
|------|---------|
| Артём (управленец/владелец) | Принимает итоги по разделам, принимает решения, ставит задачи. 80–90% продаж исторически через него лично. |
| Менеджер по продажам | Ищет объект → добивается ЛПР → переговоры → берёт на расчёт → курирует расчёт → КП → договор → аванс. Аватар: инженер со знанием сетей, бывший начальник участка, пробивной. Оклад 40 тыс. + 0.8–1.2% от суммы контракта. |
| Снабжение | Работает со спецификацией, рассылает запросы поставщикам, контролирует поставки |
| Начальник участка | На стройплощадке. Еженедельно вносит объёмы выполненных работ. Пишет заявки на материалы. Контролирует приёмку. |
| Оператор/студент | Низкая квалификация: получил счёт → загрузил → всё. ~40 тыс. руб. |
| Сметчик (девушка) | Считает стоимость работ, формирует з/п монтажников, сверяет таблицы. Один человек справляется, предела пока не видно. |
